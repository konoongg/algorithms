# SumOfSubsequences

У Маши получилось устроиться стажером в Google, и уже скоро ей выдадут первую зарплату. Чтобы не растранжирить все-все, она решила вести статистику своих доходов каждый месяц. Увы, Маша большая сладкоежка, поэтому очень много ее денег будет уходить на шоколадки. Чтобы как-то сдерживать свои порывы к сладкому, она придумала правило: покупать шоколадки только тогда, когда доход за несколько месяцев подряд равен определенному числу. Помогите Маше написать программу, которая для заданной статистики доходов и расходов по месяцам определяет суммарное количество шоколадок, которое Маша могла бы купить.

P.S. не волнуйтесь за Машу, у нее хороший метаболизм.
